Instructions for mongo db migration script
check for mongodb container  with the command docker ps and search for mongodb container is running
important- backup the database before running the script!
backup mongodb with the command mongodump --host HOSTNAME (hostname of the mongo db container )
After entering the mongo container , enter mongo shell with the command : mongo , then check if anyVision2 db exists with the command : show dbs and then command : use anyVision2 
how to check if migration went well:
before the script, run in mongo the next commands and save the results:
db.suspects.count({ignore:true})
db.detections.count({label:"ignored"})


run the command ./mongo_migration_1.19.2.sh ignoreMigration.js

after the script check by running the next commands:

db.suspects.count({ignore:true}) - Should be equal 0
db.detections.count({label:"ignored"}) - Should be equal 0
db.ignores.count() - Should be equal the same as command 1 in the before
db.suspect_groups.find(ObjectId("000000000000000000000002")).toArray()[0].title === "Low Quality Images" - Should be equal true
db.detections.find({zoom_out_images: {$exists: true}, zoom_in_images: {$exists: true}}).count() === db.detections.count() - Should be equal true
